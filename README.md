## Getting started

To run the project:

1. Clone the repo

```sh
https://github.com/rogeraugusto/design-system-react-template
```

2. Set node version with NVM or manually

```sh
nvm use # Node >= 18
```

3. Install dependencies

```sh
npm install
```

4. Run Storybook:

```sh
npm run storybook
```
